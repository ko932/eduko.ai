# **App Name**: Eduko.AI

## Core Features:

- AI Tutor - Ko AI: A witty and expert-level AI tutor that answers any academic question with clarity and provides animated video-like explanations.
- AI-Powered Tools: Suite of AI tools including Project GenX (project idea generator), AI Program Evaluator (college program advisor), and Smart Notes Generator.
- Project GenX: Generate project ideas tailored to the student's academic level and field, with a full blueprint provided after selection. Includes explanations, relevance, learning outcomes, steps, materials, execution, and timelines. Uses a reasoning tool to help students determine a suitable project difficulty.
- AI Program Evaluator: Acts as a personal admissions counselor, providing best-fit college programs based on student's stream, scores, budget, location, and goals. Evaluates programs and provides reasons for matches, admission probability, cutoff analysis, and pros/cons. Uses a reasoning tool to create personalized college suggestions based on the profile
- AI Timetable Generator: Generates a personalized weekly timetable considering subjects, strengths/weaknesses, study hours, exam dates, and lifestyle schedule. Generates a warning if the user overloads the amount of activities
- Form Central: AI-powered tool that offers step-by-step guidance on filling out forms, including eligibility criteria, age limits, fees, and warnings.
- User Dashboard: Personalized dashboard that displays a badge system, saved items, settings page, profile overview, and mock analytics such as courses completed and lessons finished.

## Style Guidelines:

- Primary color: Deep blue (#24305E), inspired by the vastness of space and the depths of knowledge, representing intelligence and focus.
- Background color: Very dark blue (#0A0D1A), offering a comfortable viewing experience.
- Accent color: Cyan (#30D5C8), providing a vibrant highlight that suggests the fusion of learning, artificial intelligence and science, all towards knowledge.
- Headline font: 'Space Grotesk', a sans-serif font with a computerized feel, for headlines and short amounts of body text. Body font: 'Inter', a grotesque-style sans-serif font for a modern, machined, objective, neutral look; suitable for body text.
- Code font: 'Source Code Pro' for displaying computer code within the platform.
- Modern, minimalist icons related to academic subjects, tools, and user interface elements, maintaining consistency and clarity.
- Subtle, elegant animations to enhance user experience, such as transitions and loading indicators.
